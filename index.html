Contribution Style

제가 기여하는 방식

빠르게 구현하는 것만큼, 실제 사용 흐름에서 문제가 없는지 반복해서 확인하는 태도를 중요하게 생각합니다.

1
사용자가 어떤 순서로 서비스를 이용할지 떠올리며 화면 이동과 입력 흐름을 반복적으로 확인합니다.
2
정상 케이스뿐 아니라 반복 클릭, 날짜 경계, 데이터 갱신 누락처럼 작은 오류 가능성도 확인합니다.
3
데이터가 저장·조회·검색·리포트로 이어지는 과정에서 값이 어긋나지 않는지 확인합니다.
4
AI 도구를 활용하더라도 결과를 직접 실행하고 검토하며, 발견한 문제를 기록합니다.
